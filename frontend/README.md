# BankShield AI Frontend

React + Vite + Tailwind CSS frontend for BankShield AI.

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

Server runs on `http://localhost:5173`

## Build

```bash
npm run build
```

Built files in `dist/`

## Features

- 🛡️ **Landing Page** - Hero section with features
- 🔐 **Authentication** - Login & Registration
- 📊 **Dashboard** - Real-time statistics
- 🚨 **Event Monitoring** - Security event feed
- 👤 **User Profiles** - Risk assessment & history
- ⚠️ **Alert Center** - Threat management
- 📈 **Reports** - Analytics & exports
- ⚙️ **Settings** - Configuration

## Pages

- `/` - Landing page
- `/login` - Login page
- `/register` - Registration page
- `/dashboard` - Main dashboard
- `/events` - Security events
- `/users/:id` - User profile
- `/alerts` - Alert center
- `/reports` - Reports
- `/settings` - Settings
